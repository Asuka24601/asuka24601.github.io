Symbol Command Symbol Command Symbol Command Symbol Command

$_$ \_ ${$ \{ $}$ \}

您可能会注意到，如果使用其中任何一个来排版垂直较大的表达式，比如
$$(\frac{a}{x} )^2$$
得到的表达式的括号的大小不对
$$(\frac{a} {x} )^2$$
